# Tracker code

Ported over to PlatformIO from [electrosew](https://github.com/mloudon/electrosew/tree/master/feather_send)

## Libs required:

- 13 : Adafruit GFX
- 124 : RadioHead
- 135 : SSA1306
- 416 : TinyGPS

To install them, go

    platformio lib install 13
    platformio lib install 124

etc.
